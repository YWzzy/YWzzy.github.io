DNS
Published inпјҡ2020-08-06 | categoryпјҡ еүҚз«Ҝ жөҸи§ҲеҷЁ

DNS (Domain Name System)пјҢ д№ҹеҸ«зҪ‘еҹҹеҗҚз§°зі»з»ҹпјҢжҳҜдә’иҒ”зҪ‘зҡ„дёҖйЎ№жңҚеҠЎгҖӮе®ғе®һиҙЁдёҠжҳҜдёҖдёӘ В еҹҹеҗҚВ  е’Ң В IPВ  зӣёдә’жҳ е°„зҡ„еҲҶеёғејҸж•°жҚ®еә“пјҢжңүдәҶе®ғпјҢжҲ‘д»¬е°ұеҸҜд»ҘйҖҡиҝҮеҹҹеҗҚжӣҙж–№дҫҝзҡ„и®ҝй—®дә’иҒ”зҪ‘гҖӮжңҖеҲқпјҢз”ұдәҺ ip й•ҝдё”йҡҫи®°пјҢйҖҡиҝҮ ip и®ҝй—®зҪ‘з«ҷдёҚж–№дҫҝгҖӮгҖӮжүҖд»ҘеҗҺжқҘйҖҡиҝҮеҸ‘жҳҺдәҶ DNS жңҚеҠЎеҷЁпјҢиҝҷдёӘж—¶еҖҷжҲ‘д»¬и®ҝй—®зҪ‘з«ҷиҫ“е…ҘзҪ‘з«ҷеҹҹеҗҚпјҢDNS жңҚеҠЎеҷЁе°ұи§ЈжһҗжҲ‘д»¬зҡ„еҹҹеҗҚдёә ipгҖӮиҝҷж ·жҲ‘д»¬е®һйҷ…и®ҝй—®зҡ„е°ұжҳҜеҜ№еә”зҡ„ ip ең°еқҖе•ҰгҖӮ
DNS жңүд»ҘдёӢзү№зӮ№:

  • еҲҶеёғејҸзҡ„
  • еҚҸи®®ж”ҜжҢҒ TCP е’Ң UDP, еёёз”Ёз«ҜеҸЈжҳҜ 53
  • жҜҸдёҖзә§еҹҹеҗҚзҡ„й•ҝеәҰйҷҗеҲ¶жҳҜ 63
  • еҹҹеҗҚжҖ»й•ҝеәҰйҷҗеҲ¶жҳҜ 253

йӮЈд№ҲпјҢд»Җд№Ҳжғ…еҶөдёӢдҪҝз”Ё TCPпјҢд»Җд№Ҳжғ…еҶөдёӢдҪҝз”Ё UDP е‘ў?
жңҖж—©зҡ„ж—¶еҖҷпјҢDNS зҡ„ UDP жҠҘж–ҮдёҠйҷҗеӨ§е°ҸжҳҜ 512 еӯ—иҠӮпјҢ жүҖд»ҘеҪ“жҹҗдёӘ response еӨ§е°Ҹи¶…иҝҮ 512 (иҝ”еӣһдҝЎжҒҜеӨӘеӨҡ)пјҢDNS жңҚеҠЎе°ұдјҡдҪҝз”Ё TCP еҚҸи®®жқҘдј иҫ“гҖӮеҗҺжқҘ DNS еҚҸи®®жү©еұ•дәҶиҮӘе·ұзҡ„ UDP еҚҸи®®пјҢDNS client еҸ‘еҮәжҹҘиҜўиҜ·жұӮж—¶пјҢеҸҜд»ҘжҢҮе®ҡиҮӘе·ұиғҪжҺҘж”¶и¶…иҝҮ 512 еӯ—иҠӮзҡ„ UDP еҢ…пјҢ иҝҷз§Қжғ…еҶөдёӢпјҢDNS иҝҳжҳҜдјҡдҪҝз”Ё UDP еҚҸи®®

еҹҹеҗҚдёҺ ip зҡ„еҜ№еә”е…ізі»пјҢиў«з§°дёәи®°еҪ•(record)пјҢеҸҜеҲҶдёәеҗ„з§Қзұ»еһӢ

  • A: AddressпјҢеҹҹеҗҚжҢҮеҗ‘зҡ„ IP ең°еқҖпјҢдёҖдёӘеҹҹеҗҚеҸҜд»ҘжңүеӨҡдёӘ A и®°еҪ•гҖӮ
  • NSпјҡName ServerпјҢдҝқеӯҳдёӢдёҖзә§еҹҹеҗҚдҝЎжҒҜзҡ„жңҚеҠЎеҷЁең°еқҖ
  • MXпјҡMail eXchangeпјҢжҺҘеҸ—з”өеӯҗйӮ®д»¶зҡ„жңҚеҠЎеҷЁең°еқҖ
  • CNAMEпјҡCanonical NameпјҢиҝ”еӣһеҸҰдёҖдёӘеҹҹеҗҚпјҢд»ӨеҪ“еүҚжҹҘиҜўеҹҹеҗҚжҢ‘еҺ»иҜҘеҹҹеҗҚпјҢеӨҡдёӘеҹҹеҗҚ->жңҚеҠЎеҷЁзҡ„жҳ е°„гҖӮ
  • PTRпјҡ Pointer RecordпјҢеҸӘз”ЁдәҺ ip ең°еқҖжҹҘиҜўеҹҹеҗҚ

DNS з”ұдёӢйқўдёүдёӘйғЁеҲҶз»„жҲҗ

  • еҗҚз§°и§ЈжһҗеҷЁпјҲresolverпјү
  • еҹҹеҗҚз©әй—ҙпјҲdomain name spaceпјү
  • еҗҚз§°жңҚеҠЎеҷЁпјҲname serverпјү

еҒҮеҰӮдҪ иҰҒи®ҝй—® baidu.comпјҢйңҖиҰҒе…ҲйҖҡиҝҮ dns зі»з»ҹжҹҘеҮәд»–зҡ„ ip ең°еқҖеҰӮ 220.181.57.216пјҢжүҚиғҪи®ҝй—®гҖӮ

dns жҹҘиҜўзҡ„иҝҮзЁӢ

dns йҖҡиҝҮеҹҹеҗҚжҹҘеҮә ipпјҲжҲ‘д»¬д»ҘжөҸи§ҲеҷЁиҫ“е…Ҙwww.example.comдёәдҫӢпјүпјҡ

  1. жЈҖжҹҘжөҸи§ҲеҷЁзј“еӯҳ(зј“еӯҳж—¶й—ҙжҜ”иҫғзҹӯпјҢй»ҳи®ӨеҸӘжңү 1 еҲҶй’ҹпјҢдё”еҸӘиғҪе®№зәі 1000 жқЎзј“еӯҳ)

  1. жЈҖжҹҘж“ҚдҪңзі»з»ҹ DNS зј“еӯҳ
  1. иҜ»еҸ–hostsж–Үд»¶
  2. жЈҖжҹҘи·Ҝз”ұеҷЁзј“еӯҳ
  3. еҰӮжһңеүҚеҮ жӯҘйғҪжІЎжІЎжүҫеҲ°пјҢдјҡеҗ‘ ISP(зҪ‘з»ңжңҚеҠЎжҸҗдҫӣе•Ҷ)зҡ„ LDNS жңҚеҠЎеҷЁжҹҘиҜў
  4. еҰӮжһң LDNS жңҚеҠЎеҷЁжІЎжүҫеҲ°пјҢдјҡеҗ‘и·ҹеҹҹеҗҚжңҚеҠЎеҷЁ(Root Server)иҜ·жұӮи§ЈжһҗпјҢеҲҶдёәд»ҘдёӢеҮ жӯҘпјҡ
    1. и·ҹжңҚеҠЎеҷЁиҝ”еӣһйЎ¶зә§еҹҹеҗҚ(TLD)жңҚеҠЎеҷЁеҰӮ.comпјҢ.cnпјҢ.org зӯүзҡ„ең°еқҖпјҢе…ЁзҗғеҸӘжңү 13 еҸ°пјҢиҜҘдҫӢеӯҗдёӯдјҡиҝ”еӣһ.com зҡ„ең°еқҖ
    2. жҺҘзқҖеҗ‘ TLD еҸ‘йҖҒиҜ·жұӮпјҢз„¶еҗҺдјҡиҝ”еӣһж¬Ўзә§еҹҹеҗҚ(SLD)жңҚеҠЎеҷЁзҡ„ең°еқҖпјҢжң¬дҫӢеӯҗдјҡиҝ”еӣһ.example зҡ„ең°еқҖ
    3. жҺҘзқҖеҗ‘ SLD еҹҹеҗҚжңҚеҠЎеҷЁйҖҡиҝҮеҹҹеҗҚжҹҘиҜўзӣ®ж Ү IPпјҢжң¬дҫӢеӯҗдјҡиҝ”еӣһwww.example.comзҡ„ең°еқҖ
    4. Local DNS Server дјҡзј“еӯҳз»“жһңпјҢе№¶иҝ”еӣһз»ҷз”ЁжҲ·пјҢзј“еӯҳеңЁзі»з»ҹдёӯгҖӮ

DNS е®үе…Ёй—®йўҳ

  1. DNS еҸҚе°„/ж”ҫеӨ§ж”»еҮ»
    еҗ‘еӨ§йҮҸејҖж”ҫ DNS жңҚеҠЎеҷЁеҸ‘йҖҒеӨ§иҢғеӣҙеҹҹеҗҚжҹҘиҜўзҡ„ DNS иҜ·жұӮпјҢе№¶е°ҶиҜҘ DNS иҜ·жұӮзҡ„жәҗ IP ең°еқҖдјӘйҖ жҲҗжғіиҰҒж”»еҮ»зҡ„зӣ®ж Ү IP ең°еқҖгҖӮз”ұдәҺиҜ·жұӮж•°жҚ®жҜ”зӣёеә”ж•°жҚ®е°Ҹеҫ—еӨҡпјҢж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҜҘжҠҖжңҜж”ҫеӨ§жҺҢжҸЎзҡ„еёҰе®Ҫиө„жәҗе’Ңж”»еҮ»жөҒйҮҸгҖӮ
  2. DDOS ж”»еҮ»еҸҜиғҪйҖ жҲҗеҹҹеҗҚи§Јжһҗзҳ«з—Ә
  3. DNS/еҹҹеҗҚеҠ«жҢҒ
    еңЁеҠ«жҢҒзҡ„зҪ‘з»ңиҢғеӣҙеҶ…жӢҰжҲӘеҹҹеҗҚи§Јжһҗзҡ„иҜ·жұӮпјҢеҲҶжһҗиҜ·жұӮзҡ„еҹҹеҗҚпјҢиҝ”еӣһеҒҮзҡ„ IP ең°еқҖжҲ–иҖ…дҪҝиҜ·жұӮеӨұеҺ»е“Қеә”гҖӮDNS еҠ«жҢҒйҖҡиҝҮзҜЎж”№ DNS жңҚеҠЎеҷЁдёҠзҡ„ж•°жҚ®иҝ”еӣһз»ҷз”ЁжҲ·дёҖдёӘй”ҷиҜҜзҡ„жҹҘиҜўз»“жһңжқҘе®һзҺ°зҡ„гҖӮ
  4. DNS жұЎжҹ“
    DNS жұЎжҹ“жҳҜдёҖз§Қи®©дёҖиҲ¬з”ЁжҲ·з”ұдәҺеҫ—еҲ°иҷҡеҒҮзӣ®ж Үдё»жңә IP иҖҢдёҚиғҪдёҺе…¶йҖҡдҝЎзҡ„ж–№жі•пјҢжҢҮзҡ„жҳҜз”ЁжҲ·и®ҝй—®дёҖдёӘең°еқҖпјҢеӣҪеҶ…зҡ„жңҚеҠЎеҷЁ(йқһ DNS)зӣ‘жҺ§еҲ°з”ЁжҲ·и®ҝй—®зҡ„е·Із»Ҹиў«ж Үи®°ең°еқҖж—¶пјҢжңҚеҠЎеҷЁдјӘиЈ…жҲҗ DNS жңҚеҠЎеҷЁеҗ‘з”ЁжҲ·еҸ‘еӣһй”ҷиҜҜзҡ„ең°еқҖзҡ„иЎҢдёәгҖӮ
    dns жұЎжҹ“дёҺ dns еҠ«жҢҒзҡ„еҢәеҲ«еңЁдәҺпјҢdns еҠ«жҢҒдҝ®ж”№дәҶ dns зҡ„и§Јжһҗз»“жһңпјҢdns жұЎжҹ“жҳҜдёҚз»ҸиҝҮ dns жңҚеҠЎеҷЁпјҢиҝ”еӣһй”ҷиҜҜдҝЎжҒҜ
  5. DNS дҝЎжҒҜй»‘е®ўиў«дҝ®ж”№

DNS дјҳеҢ–

еҸҜд»ҘзңӢеҮәпјҢdns и§ЈжһҗжҳҜдёҖдёӘжј«й•ҝзҡ„иҝҮзЁӢпјҢеҰӮдҪ•дјҳеҢ–иҝҷдёҖиҝҮзЁӢе‘ўпјҹ

  1. DNS Prefetching з”ЁжҲ·еңЁиҜ·жұӮжҹҗдёӘй“ҫжҺҘд№ӢеүҚпјҢжөҸи§ҲеҷЁе…Ҳе°қиҜ•и§ЈжһҗиҜҘй“ҫжҺҘзҡ„еҹҹеҗҚеҶҚе°Ҷе…¶иҝӣиЎҢзј“еӯҳгҖӮиҝҷж ·зңҹжӯЈиҜ·жұӮзҡ„ж—¶еҖҷе°ұдёҚйңҖиҰҒиҝӣиЎҢ DNS и§ЈжһҗгҖӮеҸҜд»Ҙ
    • еңЁжңҚеҠЎеҷЁдёӯе“Қеә”и®ҫзҪ®X-DNS-Prefetch-Controlзҡ„еҖјдёәonеҗҜеҠЁйў„и§Јжһҗ
    • HTML дёӯпјҢ<meta http-equiv="x-dns-prefetch-control" content="on">
    • еҜ№зү№е®ҡеҹҹеҗҚйў„и§Јжһҗ<link rel=вҖқdns-prefetchвҖқ href=вҖқ//fonts.googleapis.comвҖқ>
  2. еҹҹеҗҚж”¶ж•ӣ
    е»әи®®е°ҶйқҷжҖҒиө„жәҗеҸӘж”ҫеңЁдёҖдёӘеҹҹеҗҚдёӢйқўпјҢеҸҜд»Ҙжңүж•ҲеҮҸе°‘ dns зҡ„иҜ·жұӮ
  3. httpdns еҹәдәҺ Http еҚҸи®®еҗ‘ HTTPDNS жңҚеҠЎеҷЁеҸ‘йҖҒеҹҹеҗҚи§ЈжһҗиҜ·жұӮпјҢжӣҝд»ЈдәҶеҹәдәҺ DNS еҚҸи®®еҗ‘иҝҗиҗҘе•Ҷ Local DNS еҸ‘иө·и§ЈжһҗиҜ·жұӮзҡ„дј з»ҹж–№ејҸпјҢеҸҜд»ҘйҒҝе…ҚиҝҗиҗҘе•Ҷзҡ„еҹҹеҗҚеҠ«жҢҒе’ҢиҝӣиЎҢзІҫеҮҶи°ғеәҰгҖӮиҝҷиҝҮзЁӢеҲҶдёәдёӨжӯҘ
    1. е®ўжҲ·з«ҜзӣҙжҺҘи®ҝй—® HttpDNS жҺҘеҸЈпјҢиҺ·еҸ–дёҡеҠЎеңЁеҹҹеҗҚй…ҚзҪ®з®ЎзҗҶзі»з»ҹдёҠй…ҚзҪ®зҡ„и®ҝй—®е»¶иҝҹжңҖдјҳзҡ„ IPгҖӮпјҲеҹәдәҺе®№зҒҫиҖғиҷ‘пјҢиҝҳжҳҜдҝқз•ҷж¬ЎйҖүдҪҝз”ЁиҝҗиҗҘе•Ҷ LocalDNS и§ЈжһҗеҹҹеҗҚзҡ„ж–№ејҸпјү
    2. е®ўжҲ·з«Ҝеҗ‘иҺ·еҸ–еҲ°зҡ„ IP еҗҺе°ұеҗ‘зӣҙжҺҘеҫҖжӯӨ IP еҸ‘йҖҒдёҡеҠЎеҚҸи®®иҜ·жұӮгҖӮд»Ҙ Http иҜ·жұӮдёәдҫӢпјҢйҖҡиҝҮеңЁ header дёӯжҢҮе®ҡ host еӯ—ж®өпјҢеҗ‘ HttpDNS иҝ”еӣһзҡ„ IP еҸ‘йҖҒж ҮеҮҶзҡ„ Http иҜ·жұӮеҚіеҸҜгҖӮ

еҸӮиҖғж–Үз«
