1. еҝ«йҖҹе…Ҙй—ЁпјҢеҹ№е…»е…ҙи¶Ј

еҲҡејҖе§ӢеӯҰд№ еүҚз«Ҝж—¶пјҢдёҖе®ҡиҰҒд»Ҙеҹ№е…»е…ҙи¶ЈгҖҒеҝ«йҖҹе…Ҙй—Ёдёәзӣ®ж ҮгҖӮжҲ‘е…Ҙй—ЁеүҚз«Ҝж—¶пјҢз”ұдәҺжІЎжңүеҹәзЎҖгҖҒе•ҘйғҪдёҚжҮӮпјҢеӣ жӯӨйҖүжӢ©еңЁзәҝе®һжҲҳж•ҷзЁӢзҪ‘з«ҷжқҘеӯҰд№ пјҢи·ҹзқҖж•ҷзЁӢжЎҲдҫӢдёҖиЎҢиЎҢжҠҠд»Јз Ғж•ІдёӢжқҘпјҢе®һж—¶жөҸи§ҲжҳҫзӨәж•ҲжһңпјҢеҫҲиҪ»жқҫең°е°ұжҠҠ HTMLгҖҒCSSгҖҒJavaScript зҡ„иҜӯжі•еҹәзЎҖиҝҮдәҶдёҖйҒҚгҖӮдҪҶжҳҜиҝҷж—¶пјҢжҲ‘еҜ№зңӢиҝҮзҡ„зҹҘиҜҶзӮ№жІЎжңүд»Җд№ҲеҚ°иұЎпјҢеҹәжң¬жҳҜзңӢдәҶе°ұеҝҳпјҢеҪ“жҲ‘жғіиҰҒиҮӘе·ұеҶҷдёҖдёӘзҪ‘з«ҷпјҢд№ҹж №жң¬ж— д»ҺдёӢжүӢгҖӮ
дәҺжҳҜпјҢжҲ‘ејҖе§ӢиҜ•зқҖжӯ»и®°зЎ¬иғҢд»Јз ҒпјҢз»“жһңеҸ‘зҺ°иғҢдәҶд№ӢеҗҺиҝҳжҳҜдёҚдјҡеҶҷгҖӮйӮЈе№Іи„ҶдёҖдёҚеҒҡдәҢдёҚдј‘пјҢзӣҙжҺҘжү“ејҖзј–иҫ‘еҷЁе’Ңж•ҷзЁӢзҪ‘з«ҷпјҢжҠҠд№ӢеүҚеңЁзәҝе®һжҲҳж—¶ж•ІиҝҮзҡ„д»Јз ҒеӨҚеҲ¶зІҳиҙҙеҲ°зј–иҫ‘еҷЁдёӯпјҢз„¶еҗҺеҶҚдҝ®ж”№дҝқеӯҳпјҢеҲ°жөҸи§ҲеҷЁдёӯжү“ејҖж–Үд»¶гҖӮиҝҷж ·иҷҪз„¶иҰҒеңЁзј–иҫ‘еҷЁе’ҢжөҸи§ҲеҷЁдёӯжқҘеӣһеҲҮжҚўпјҢдҪҶиө·з ҒиғҪзңӢеҲ°иҮӘе·ұзҡ„зҪ‘йЎөж–Үд»¶иҝҗиЎҢзҡ„ж•ҲжһңдәҶгҖӮеҗҺжқҘпјҢжҲ‘еҲ©з”ЁжөҸи§ҲеҷЁејҖеҸ‘иҖ…е·Ҙе…·жҸҗеҚҮдәҶиҮӘе·ұејҖеҸ‘зҪ‘з«ҷзҡ„и°ғиҜ•ж•ҲзҺҮпјҢж•ІеҮәзҡ„д»Јз ҒиҫҫдёҚеҲ°ж»Ўж„Ҹзҡ„ж•Ҳжһңж—¶пјҢе°ұеҶҚеҺ»жҹҘпјҢеҶҚж”№пјҢеҶҚжөҸи§ҲпјҢжңҖеҗҺз»ҲдәҺе®ҢжҲҗдәҶдёҖдёӘз•ҷиЁҖжқҝзҪ‘з«ҷпјҒ
йҖҡиҝҮиҝҷд»¶дәӢпјҢжҲ‘жҳҺзҷҪдәҶдёҖдёӘйҒ“зҗҶпјҢж—ўз„¶и®°дёҚдҪҸзҹҘиҜҶпјҢйӮЈе°ұе…ҲеҲ«ејәиЎҢи®°еҝҶпјҢдёҠжүӢеӨҡеҶҷпјҢеҝҳдәҶе°ұеҺ»жҹҘдёҚе°ұеҘҪдәҶпјҹдёҚиҰҒжӢ…еҝғиҮӘе·ұзҹҘиҜҶдёҚеӨҹеҒҡдёҚеҮәжқҘпјҢеҸӘиҰҒдҪ еҺ»иҜ•пјҢйғҪдёҖе®ҡиғҪе®һзҺ°гҖӮ

2. еӨҡзңӢеӨҡеҶҷпјҢе·©еӣәеҹәзЎҖ

еңЁиҝҷз§Қж“ҚдҪңдёӢпјҢжҲ‘иҷҪз„¶иғҪдҪҝз”ЁеҹәзЎҖдёүд»¶еҘ—ејҖеҸ‘з®Җжҳ“зҪ‘з«ҷдәҶпјҢдҪҶеҒҡзҡ„еҫҲж…ўпјҢиҝҳиҖҒеҮәй—®йўҳгҖӮдәҺжҳҜпјҢжҲ‘ејҖе§Ӣе®Ңж•ҙең°зңӢдәҶеҮ жң¬еүҚз«Ҝе…Ҙй—Ёе’ҢйЎ№зӣ®е®һжҲҳд№ҰзұҚпјҢжңүдёүд»¶еҘ—е…Ҙй—Ёзҡ„пјҢд№ҹжңүе“Қеә”ејҸзҪ‘з«ҷи®ҫи®Ўзҡ„пјҢиҝҳжңү JavaScript з»јеҗҲи®Іи§Јзҡ„пјҢд»ҘеҸҠдёҖеҘ—е®Ңж•ҙзҡ„и§Ҷйў‘ж•ҷзЁӢпјҢйҖҡиҝҮиҝҷдәӣжқҘеӨҚд№ е·©еӣәд№ӢеүҚеӯҰиҝҮзҡ„зҹҘиҜҶпјҢжү“еҘҪдәҶеҹәзЎҖпјҢе№¶дё”дәҶи§ЈдәҶжӣҙеӨҡе®һжҲҳж•ҷзЁӢдёҠжІЎи§ҒеҲ°зҡ„зҹҘиҜҶгҖӮ
зңӢд№Ұе’Ңи§Ҷйў‘зҡ„иҝҮзЁӢдёӯпјҢжҲ‘и·ҹзқҖеҒҡдәҶеҮ дёӘз®Җжҳ“зҡ„зҪ‘з«ҷпјҢд№ҹжҳҜиҫ№зңӢиҫ№еҒҡгҖӮеңЁиҝҷд№ӢеҗҺпјҢжҲ‘еҸ‘зҺ°иҮӘе·ұжёҗжёҗең°иғҪеӨҹи„ұзҰ»ж–ҮжЎЈжқҘеҶҷзҪ‘з«ҷдәҶпјҒ

3. жЎҶжһ¶еӯҰд№ пјҢеӯҰд»ҘиҮҙз”Ё

дҪҶжҳҜпјҢжҖ»ж„ҹи§үејҖеҸ‘зҪ‘з«ҷж—¶пјҢд»Җд№ҲйғҪиҰҒиҮӘе·ұеҶҷпјҢйҮҚеӨҚзҡ„д»Јз ҒеҶҷдёҖеӨ§е ҶпјҢеҫҲйә»зғҰгҖӮдәҺжҳҜпјҢжҲ‘иҜ•зқҖдёҠзҪ‘жҗңдәҶи§ЈеҶіиҝҷдәӣй—®йўҳзҡ„ж–№жі•пјҢеҸ‘зҺ°еҸҜд»Ҙз”Ёз»„д»¶еә“еҮҸе°‘йҮҚеӨҚзј–еҶҷ HTML е’Ң CSS д»Јз ҒпјҢдәҺжҳҜеӯҰдәҶ LayUIпјҢе…¶е®һе’Ңд№ӢеүҚдёҖж ·зҡ„пјҢе°ұжҳҜжү“ејҖе®ҳзҪ‘жүҫеҲ°иҰҒз”Ёзҡ„з»„д»¶пјҢз„¶еҗҺеӨҚеҲ¶еҲ°йЎ№зӣ®дёӯдҝ®ж”№е°ұиЎҢдәҶпјҢеҶҷзҡ„еӨҡдәҶз«ҹз„¶иҝҳзңҹи®°дҪҸдәҶдёҖдәӣпјҲиҷҪз„¶иҝҷдёӘжІЎеҝ…иҰҒи®°пјүгҖӮ
з•Ңйқўе’Ңж ·ејҸд»Јз Ғз®ҖеҢ–дәҶпјҢдҪҶжҳҜ JavaScript еҶҷзҡ„жҲ‘дҫқж—§еӨҙз–јгҖӮдәҺжҳҜжҲ‘еҶіе®ҡиҝӣе…ҘдёӢдёҖдёӘйҳ¶ж®өзҡ„еӯҰд№ пјҢеҪ“ж—¶е…ҲзңӢдәҶжң¬д№ҰеҸ«гҖҠй”ӢеҲ©зҡ„ jQueryгҖӢжқҘеӯҰд№  jQueryпјҢеӯҰдјҡд№ӢеҗҺдҪҝз”Ёе®ғи¶Ғзғӯжү“й“ҒеҒҡдәҶеҮ дёӘзҪ‘з«ҷпјҢеҸ‘зҺ°зҡ„зЎ®иғҪеӨ§еӨ§з®ҖеҢ–з№Ғзҗҗзҡ„ JavaScript DOM ж“ҚдҪңпјҢи®©жҲ‘е°қеҲ°дәҶз”ңеӨҙпјҢеҸҲжңүдәҶз»§з»ӯеӯҰд№ еүҚз«Ҝзҡ„еҠЁеҠӣгҖӮ
еңЁйӮЈд№ӢеҗҺеӨ§жҰӮеҚҠе№ҙпјҢжҲ‘ејҖе§ӢеӯҰд№ дё»жөҒеүҚз«ҜжЎҶжһ¶ VueпјҢд№ҹжҳҜи·ҹзқҖи§Ҷйў‘ж•ҷзЁӢеӯҰпјҢе№¶дё”з»“еҗҲ ElementUI з»„д»¶еә“ејҖеҸ‘дәҶдёҖдёӘе“Қеә”ејҸзҪ‘з«ҷгҖӮдёҚеҫ—дёҚиҜҙпјҢд№ жғҜдәҶз”ЁеҺҹз”ҹ JavaScript е’Ң jQuery ејҖеҸ‘еҗҺпјҢеҶҚеҺ»з”Ё Vue ејҖеҸ‘пјҢзңҹзҡ„жҳҜзҲҪйЈһдәҶпјҢе°ҸжңүжҲҗе°ұж„ҹгҖӮ

4. ж №жҚ®йңҖиҰҒпјҢиҮӘдё»еӯҰд№ 

еңЁйӮЈд№ӢеҗҺпјҢжҲ‘е°ұжҳҜйҖҡиҝҮдёҚж–ӯең°еҒҡйЎ№зӣ®жқҘз§ҜзҙҜз»ҸйӘҢпјҢж—ўз„¶зј–зЁӢзҹҘиҜҶеӨӘеӨҡеӯҰдёҚе®ҢпјҢйӮЈжҲ‘е°ұйҡҸйңҖйҡҸеӯҰпјҢйҖҗжёҗжӢ“е®ҪиҮӘе·ұзҡ„зҹҘиҜҶйқўгҖӮдёҫдёӘдҫӢеӯҗпјҢжҲ‘иҰҒеҒҡдёҖдёӘз§»еҠЁз«Ҝ H5 йЎөйқўпјҢйӮЈе°ұеҺ»жҗң Vue зҡ„з§»еҠЁз«Ҝз»„д»¶еә“пјҢеӯҰдјҡдәҶ MintUIгҖҒVantUI зӯүпјӣйЎ№зӣ®дёӯиҰҒеӨ„зҗҶж—ҘжңҹпјҢйӮЈжҲ‘е°ұеҺ»жҗңж—ҘжңҹеӨ„зҗҶеә“пјҢеӯҰдјҡдәҶ Moment.jsпјӣйЎ№зӣ®йҮҚеӨҚж–Үд»¶е’Ңд»Јз ҒеӨӘеӨҡпјҢжҲ‘е°ұеҺ»зҪ‘дёҠжҗңдёҖдәӣз»„д»¶еҢ–гҖҒжЁЎеқ—еҢ–зҡ„жңҖдҪіе®һи·өпјӣйЎ№зӣ®д»Јз ҒеӨӘеӨ§гҖҒеҠ иҪҪеӨӘж…ўпјҢжҲ‘е°ұеҺ»зҪ‘дёҠжҗңдёҖдәӣеүҚз«ҜеҺӢзј©гҖҒжҖ§иғҪдјҳеҢ–зҡ„ж–№ејҸпјӣиҰҒжҠҠејҖеҸ‘еҘҪзҡ„йЎ№зӣ®еҸҳжҲҗ APPпјҢйӮЈжҲ‘е°ұеҺ»зҪ‘дёҠжҗңеҲ°дәҶ Cordova зӯүжү“еҢ…е·Ҙе…·пјӣи·ҹе…¶д»–еҗҢеӯҰдёҖиө·еҒҡйЎ№зӣ®ж—¶жҜ”иҫғж··д№ұгҖҒејҖеҸ‘ж•ҲзҺҮдҪҺпјҢжҲ‘е°ұеҺ»жҗңдәҶдёҖдәӣд»Јз Ғи§„иҢғгҖҒеүҚз«Ҝе·ҘзЁӢеҢ–зӯүпјӣйҒҮеҲ° Bug ж—¶пјҢд№ҹжҳҜиҮӘе·ұеҺ»зҪ‘дёҠжҗңзҙўи§ЈеҶіж–№жЎҲпјҢе№¶дё”иҝҳиҰҒзңӢзңӢжңүжІЎжңүзҹҘиҜҶзӮ№жҳҜжҲ‘йҒ—жјҸзҡ„пјҢжҲ‘дјҡеҶҚеҺ»ејҘиЎҘеӯҰд№ гҖӮйҖҡиҝҮдёҚж–ӯе®һи·өгҖҒдёҚж–ӯйҒҮеҲ°й—®йўҳе’ҢиҮӘдё»и§ЈеҶій—®йўҳпјҢжҲ‘е§Ӣз»ҲдҝқжҢҒзқҖеӯҰд№ зҡ„з§ҜжһҒжҖ§пјҢеӯҰеҲ°дәҶи¶ҠжқҘи¶ҠеӨҡеүҚз«Ҝзҡ„зҹҘиҜҶпјҢе№¶йҖҡиҝҮеҪ’зәіжҖ»з»“жқҘеҠ ж·ұеҚ°иұЎгҖӮ
