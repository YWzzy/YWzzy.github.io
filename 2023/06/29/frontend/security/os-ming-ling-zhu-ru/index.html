жңүж—¶пјҢжӮЁжҺ§еҲ¶зҡ„иҫ“е…ҘдјҡеҮәзҺ°еңЁеҺҹе§Ӣе‘Ҫд»Өзҡ„еј•еҸ·еҶ…гҖӮеңЁиҝҷз§Қжғ…еҶөдёӢпјҢжӮЁйңҖиҰҒеңЁдҪҝз”ЁеҗҲйҖӮзҡ„ shell е…ғеӯ—з¬ҰжіЁе…Ҙж–°е‘Ҫд»Өд№ӢеүҚз»Ҳжӯўеј•з”Ёзҡ„дёҠдёӢж–ҮпјҲдҪҝз”ЁвҖқжҲ–вҖҷпјүгҖӮ

еҰӮдҪ•йҳІжӯўж“ҚдҪңзі»з»ҹе‘Ҫд»ӨжіЁе…Ҙж”»еҮ»

иҝ„д»ҠдёәжӯўпјҢд»ҘйҳІжӯўж“ҚдҪңзі»з»ҹе‘Ҫд»ӨжіЁе…ҘжјҸжҙһзҡ„жңҖжңүж•Ҳзҡ„ж–№жі•жҳҜд»ҺжңӘи°ғеҮәд»Һеә”з”ЁеұӮд»Јз Ғ OS е‘Ҫд»ӨгҖӮеңЁеҮ д№ҺжүҖжңүжғ…еҶөдёӢпјҢеӯҳеңЁдҪҝз”Ёе®үе…Ёе№іеҸ° API е®һзҺ°жүҖйңҖеҠҹиғҪзҡ„жӣҝд»Јж–№ејҸгҖӮ
еҰӮжһңи®ӨдёәдҪҝз”Ёз”ЁжҲ·жҸҗдҫӣзҡ„иҫ“е…Ҙи°ғз”Ёж“ҚдҪңзі»з»ҹе‘Ҫд»ӨдёҚеҸҜйҒҝе…ҚпјҢеҲҷеҝ…йЎ»жү§иЎҢејәиҫ“е…ҘйӘҢиҜҒгҖӮжңүж•ҲйӘҢиҜҒзҡ„дёҖдәӣдҫӢеӯҗеҢ…жӢ¬пјҡ

  • йӘҢиҜҒй’ҲеҜ№е…Ғи®ёеҖјзҡ„зҷҪеҗҚеҚ•гҖӮ
  • йӘҢиҜҒиҫ“е…ҘжҳҜеҗҰдёәж•°еӯ—гҖӮ
  • йӘҢиҜҒиҫ“е…Ҙд»…еҢ…еҗ«еӯ—жҜҚж•°еӯ—еӯ—з¬ҰпјҢжІЎжңүе…¶д»–иҜӯжі•жҲ–з©әж јгҖӮ

ж°ёиҝңдёҚиҰҒиҜ•еӣҫйҖҡиҝҮиҪ¬д№ү shell е…ғеӯ—з¬ҰжқҘжё…зҗҶиҫ“е…ҘгҖӮе®һйҷ…дёҠпјҢиҝҷеӨӘе®№жҳ“еҮәй”ҷе№¶дё”е®№жҳ“иў«зҶҹз»ғзҡ„ж”»еҮ»иҖ…з»•иҝҮгҖӮ
