React з»„д»¶еҢ–ејҖеҸ‘
Published inпјҡ2023-07-14 | categoryпјҡ еүҚз«Ҝ React

дёҖгҖҒд»Җд№ҲжҳҜз»„д»¶еҢ–

еңЁеӨ„зҗҶдёҖдёӘеӨҚжқӮзҡ„й—®йўҳж—¶пјҢжҲ‘д»¬йҖ»иҫ‘жҖқз»ҙиғҪеҠӣжҳҜжңүйҷҗзҡ„пјҢдёҚеӨӘеҸҜиғҪдёҖж¬ЎжҖ§жҗһе®ҡжүҖжңүзҡ„еҶ…е®№пјҢйӮЈжҲ‘д»¬е°ұеҸҜд»ҘеҜ№й—®йўҳиҝӣиЎҢжӢҶи§ЈпјҢжӢҶи§ЈжҲҗдёҖдёӘдёӘе°Ҹзҡ„й—®йўҳпјҢжңҖеҗҺе°Ҷе…¶ж”ҫе…Ҙж•ҙдҪ“дёӯпјҢиҝҷе°ұжҳҜеҲҶжІ»зҡ„жҖқжғігҖӮ

еҲҶиҖҢжІ»д№ӢжҳҜиҪҜд»¶е·ҘзЁӢзҡ„йҮҚиҰҒжҖқжғіпјҢжҳҜеӨҚжқӮзі»з»ҹејҖеҸ‘е’Ңз»ҙжҠӨзҡ„еҹәзҹіпјҢиҖҢеүҚз«Ҝзӣ®еүҚзҡ„жЁЎеқ—еҢ–е’Ңз»„д»¶еҢ–йғҪжҳҜеҹәдәҺеҲҶиҖҢжІ»д№Ӣзҡ„жҖқжғігҖӮ

еҰӮжһңжҲ‘д»¬е°ҶдёҖдёӘйЎөйқўзҡ„жүҖжңүзҡ„еӨ„зҗҶйҖ»иҫ‘йғҪж”ҫеңЁдёҖиө·пјҢйӮЈеӨ„зҗҶиө·жқҘе°ұдјҡйқһеёёз№ҒжқӮпјҢе№¶дё”д№ҹдёҚеҲ©дәҺеҗҺз»ӯзҡ„з®ЎзҗҶд»ҘеҸҠжү©еұ•гҖӮжҲ‘д»¬еҸҜд»Ҙе°ҶйЎөйқўжӢҶеҲҶжҲҗдёҖдёӘдёӘе°Ҹзҡ„еҠҹиғҪжЁЎеқ—пјҢжҜҸдёӘеҠҹиғҪжЁЎеқ—е®ҢжҲҗеұһдәҺиҝҷйғЁеҲҶзӢ¬з«Ӣзҡ„еҠҹиғҪпјҢиҝҷж ·д№ӢеҗҺж•ҙдёӘйЎөйқўзҡ„з®ЎзҗҶе’Ңз»ҙжҠӨеҸҳеҫ—еҫҲе®№жҳ“дәҶгҖӮ

жҲ‘д»¬йңҖиҰҒйҖҡиҝҮз»„д»¶еҢ–зҡ„жҖқжғіжқҘжҖқиҖғж•ҙдёӘеә”з”ЁзЁӢеәҸпјҡ
в—Ҹ е°ҶдёҖдёӘе®Ңж•ҙзҡ„йЎөйқўеҲҶжҲҗеҫҲеӨҡдёӘз»„д»¶пјӣ
в—Ҹ жҜҸдёӘз»„д»¶йғҪз”ЁдәҺе®һзҺ°йЎөйқўзҡ„дёҖдёӘеҠҹиғҪеқ—пјӣ
в—Ҹ жҜҸдёҖдёӘз»„д»¶еҸҲеҸҜд»ҘиҝӣиЎҢз»ҶеҲҶпјӣ
в—Ҹ з»„д»¶жң¬иә«еҸҲеҸҜд»ҘеңЁеӨҡдёӘең°ж–№иҝӣиЎҢеӨҚз”Ёпјӣ

дәҢгҖҒReact з»„д»¶еҢ–

з»„д»¶еҢ–жҳҜ React жҳҜж ёеҝғжҖқжғіпјҢReact зҡ„з»„д»¶зӣёеҜ№дәҺ Vue жӣҙеҠ зҒөжҙ»е’ҢеӨҡж ·пјҢжҢүз…§дёҚеҗҢзҡ„ж–№ејҸеҸҜд»ҘеҲҶжҲҗеҫҲеӨҡзұ»з»„д»¶пјҡ
в—Ҹ ж №жҚ®з»„д»¶зҡ„е®ҡд№үж–№ејҸпјҢеҸҜд»ҘеҲҶдёәпјҡ**еҮҪж•°з»„д»¶(Functional Component )е’Ңзұ»з»„д»¶(Class Component)пјӣ
в—Ҹ ж №жҚ®з»„д»¶еҶ…йғЁжҳҜеҗҰжңүзҠ¶жҖҒйңҖиҰҒз»ҙжҠӨпјҢеҸҜд»ҘеҲҶжҲҗпјҡж— зҠ¶жҖҒз»„д»¶(Stateless Component )е’ҢжңүзҠ¶жҖҒз»„д»¶(Stateful Component)пјӣ
в—Ҹ ж №жҚ®з»„д»¶зҡ„дёҚеҗҢиҒҢиҙЈпјҢеҸҜд»ҘеҲҶжҲҗпјҡеұ•зӨәеһӢз»„д»¶(Presentational Component)е’Ңе®№еҷЁеһӢз»„д»¶(Container Component)**пјӣ

иҝҷдәӣжҰӮеҝөжңүеҫҲеӨҡйҮҚеҸ пјҢдҪҶжҳҜе®ғд»¬жңҖдё»иҰҒжҳҜе…іжіЁж•°жҚ®йҖ»иҫ‘е’Ң UI еұ•зӨәзҡ„еҲҶзҰ»пјҡ
в—Ҹ еҮҪж•°з»„д»¶гҖҒж— зҠ¶жҖҒз»„д»¶гҖҒеұ•зӨәеһӢз»„д»¶дё»иҰҒе…іжіЁ UI зҡ„еұ•зӨәпјӣ
в—Ҹ зұ»з»„д»¶гҖҒжңүзҠ¶жҖҒз»„д»¶гҖҒе®№еҷЁеһӢз»„д»¶дё»иҰҒе…іжіЁж•°жҚ®йҖ»иҫ‘пјӣ
еҪ“з„¶иҝҳжңүеҫҲеӨҡз»„д»¶зҡ„е…¶д»–жҰӮеҝөпјҡжҜ”еҰӮејӮжӯҘз»„д»¶гҖҒй«ҳйҳ¶з»„д»¶зӯү

1. зұ»з»„д»¶

зұ»з»„д»¶зҡ„е®ҡд№үжңүеҰӮдёӢиҰҒжұӮпјҡ
в—Ҹ з»„д»¶зҡ„еҗҚз§°жҳҜеӨ§еҶҷеӯ—з¬ҰејҖеӨҙпјҲж— и®әзұ»з»„д»¶иҝҳжҳҜеҮҪж•°з»„д»¶пјү
в—Ҹ зұ»з»„д»¶йңҖиҰҒз»§жүҝиҮӘ React.Component
в—Ҹ зұ»з»„д»¶еҝ…йЎ»е®һзҺ°renderеҮҪж•°

еңЁ ES6 д№ӢеүҚпјҢеҸҜд»ҘйҖҡиҝҮcreate-react-class жЁЎеқ—жқҘе®ҡд№үзұ»з»„д»¶пјҢдҪҶжҳҜзӣ®еүҚе®ҳзҪ‘е»әи®®жҲ‘д»¬дҪҝз”ЁES6зҡ„classзұ»е®ҡд№үгҖӮ

зұ»з»„д»¶е®ҡд№ү

дҪҝз”Ёclassе®ҡд№үдёҖдёӘз»„д»¶пјҡ
в—Ҹ constructorжҳҜеҸҜйҖүзҡ„пјҢжҲ‘д»¬йҖҡеёёеңЁconstructorдёӯеҲқе§ӢеҢ–дёҖдәӣж•°жҚ®пјӣ
в—Ҹ this.stateдёӯз»ҙжҠӨзҡ„е°ұжҳҜжҲ‘д»¬з»„д»¶еҶ…йғЁзҡ„ж•°жҚ®пјӣ
в—Ҹ render() ж–№жі•жҳҜ class з»„д»¶дёӯе”ҜдёҖеҝ…йЎ»е®һзҺ°зҡ„ж–№жі•пјӣ

зұ»з»„д»¶зү№зӮ№

  1. зҠ¶жҖҒ state жҳҜеңЁ constructor дёӯеҲқе§ӢеҢ–зҡ„пјӣ
    2. жҲҗе‘ҳеҮҪж•°дёҚдјҡиҮӘеҠЁз»‘е®ҡ thisпјҢйңҖиҰҒејҖеҸ‘иҖ…жүӢеҠЁз»‘е®ҡпјҢеҗҰеҲҷ this дёҚиғҪиҺ·еҸ–еҪ“еүҚз»„д»¶е®һдҫӢеҜ№иұЎгҖӮReact дёӯеҸҜд»Ҙз”Ёд»ҘдёӢдёүз§Қж–№жі•жүӢеҠЁз»‘е®ҡ this пјҡ
    в—Ҹ еҸҜд»ҘеңЁжһ„йҖ еҮҪж•°дёӯе®ҢжҲҗз»‘е®ҡпјӣ
    в—Ҹ еҸҜд»ҘеңЁи°ғз”Ёж—¶дҪҝз”Ё method.bind(this) жқҘе®ҢжҲҗз»‘е®ҡпјӣ
    в—Ҹ еҸҜд»ҘдҪҝз”Ёз®ӯеӨҙеҮҪж•°жқҘз»‘е®ҡгҖӮ

йҖҡиҝҮ state зҡ„еҸҳеҢ–пјҢз»„д»¶зҡ„ UI д№ҹдјҡйҡҸд№ӢйҮҚж–°жёІжҹ“пјҢеӣ жӯӨжҲ‘д»¬дёҚиғҪжҠҠжүҖжңүеҸҳйҮҸйғҪж”ҫеҲ° state дёӯпјҢдёҚз„¶дјҡйҖ жҲҗдёҖе®ҡзҡ„жҖ§иғҪжҚҹиҖ—гҖӮ**д»ҘдёӢжғ…еҶөйғҪдёҚеә”иҜҘдҪңдёәдёҖдёӘзҠ¶жҖҒж”ҫеҲ° **state** дёӯпјҡ **
в—Ҹ еҰӮжһңжҳҜйҖҡиҝҮ props иҺ·еҸ–пјӣ
в—Ҹ еҰӮжһңиҜҘеҸҳйҮҸдёҚеңЁ render дёӯдҪҝз”Ёпјӣ
в—Ҹ еҰӮжһңж•ҙдёӘиҝҮзЁӢдёӯйғҪдёҚдјҡеҸ‘з”ҹеҸҳеҢ–пјӣ
в—Ҹ еҰӮжһңиҜҘеҸҳйҮҸеҸҜд»ҘйҖҡиҝҮ state е’Ң props дёӨиҖ…и®Ўз®—еҫ—еҮәгҖӮ

2. еҮҪж•°з»„д»¶

еҮҪж•°з»„д»¶жҳҜдҪҝз”ЁfunctionжқҘиҝӣиЎҢе®ҡд№үзҡ„еҮҪж•°пјҢеҸӘжҳҜиҝҷдёӘеҮҪж•°дјҡиҝ”еӣһе’Ңзұ»з»„д»¶дёӯrenderеҮҪж•°иҝ”еӣһдёҖж ·зҡ„еҶ…е®№гҖӮ

еҮҪж•°з»„д»¶жңүиҮӘе·ұзҡ„зү№зӮ№пјҡ
в—Ҹ жІЎжңүз”ҹе‘Ҫе‘ЁжңҹпјҢд№ҹдјҡиў«жӣҙж–°е№¶жҢӮиҪҪпјҢдҪҶжҳҜжІЎжңүз”ҹе‘Ҫе‘ЁжңҹеҮҪж•°
в—Ҹ жІЎжңүthis(з»„д»¶е®һдҫӢпјү
в—Ҹ жІЎжңүеҶ…йғЁзҠ¶жҖҒпјҲstateпјү

еҮҪж•°ејҸеҲӣе»әзҡ„з»„д»¶д»Јз Ғз®ҖжҙҒпјҢдё“жіЁдәҺ renderпјҢдё”з»„д»¶дёҚйңҖиҰҒиў«е®һдҫӢеҢ–пјҢж•ҙдҪ“жёІжҹ“жҖ§иғҪеҫ—еҲ°дәҶжҸҗеҚҮпјҢдё”и§Ҷеӣҫе’Ңж•°жҚ®и§ЈиҖҰеҲҶзҰ»пјҢиҫ“еҮәеҸӘеҸ–еҶідәҺиҫ“е…ҘгҖӮдҪҶжҳҜпјҢе®ғж— жі•жӢҘжңүиҮӘе·ұзҡ„ stateпјҢеҸӘиғҪйҖҡиҝҮ props иҺ·еҸ–еұһжҖ§еҶ…е®№е№¶е®һзҺ°з»„д»¶зҡ„жӣҙж–°пјҢж— з”ҹе‘Ҫе‘ЁжңҹгҖӮ

дёӢйқўжқҘе®ҡд№үдёҖдёӘеҮҪж•°з»„д»¶пјҡ

иҜҙе®ҢиҝҷдёӨз§Қз»„д»¶пјҢжҲ‘д»¬жқҘзңӢдёҖдёӢrenderеҮҪж•°зҡ„иҝ”еӣһеҖјгҖӮеҪ“ render иў«и°ғз”Ёж—¶пјҢе®ғдјҡжЈҖжҹҘ this.props е’Ң this.state зҡ„еҸҳеҢ–е№¶иҝ”еӣһд»ҘдёӢзұ»еһӢд№ӢдёҖпјҡ
**пјҲ1пјү React е…ғзҙ пјҡ **йҖҡеёёйҖҡиҝҮ JSX еҲӣе»әгҖӮ дҫӢеҰӮпјҡ<div /> дјҡиў« React жёІжҹ“дёә DOM иҠӮзӮ№пјҢ<MyComponent /> дјҡиў« React жёІжҹ“дёәиҮӘе®ҡд№үз»„д»¶гҖӮ <div /> е’Ң <MyComponent /> йғҪжҳҜ React е…ғзҙ гҖӮ
пјҲ2пјүж•°з»„жҲ– fragmentsпјҡдҪҝеҫ— render ж–№жі•еҸҜд»Ҙиҝ”еӣһеӨҡдёӘе…ғзҙ гҖӮ
пјҲ3пјүPortalsпјҡеҸҜд»ҘжёІжҹ“еӯҗиҠӮзӮ№еҲ°дёҚеҗҢзҡ„ DOM еӯҗж ‘дёӯгҖӮ
пјҲ4пјүеӯ—з¬ҰдёІжҲ–ж•°еҖјзұ»еһӢпјҡе®ғд»¬еңЁ DOM дёӯдјҡиў«жёІжҹ“дёәж–Үжң¬иҠӮзӮ№
пјҲ5пјүеёғе°”зұ»еһӢжҲ– nullпјҡд»Җд№ҲйғҪдёҚжёІжҹ“гҖӮ
